# 💛 Insurance Cost Prediction System

A professional Machine Learning web application that predicts medical insurance charges using Linear Regression, deployed with Streamlit.

---

## 📌 Project Overview

This system analyzes demographic and health-related attributes to estimate individual medical insurance costs.  
The model is trained using supervised learning (Linear Regression) and deployed as an interactive web application.

---

## 🚀 Live Features

- Real-time insurance cost prediction
- R² accuracy score display
- Clean and structured layout
- Input validation with sliders and dropdowns

---

## 📊 Model Information

- **Algorithm Used:** Linear Regression  
- **Learning Type:** Supervised Learning  
- **Problem Type:** Regression  
- **Evaluation Metric:** R² Score  

Typical R² Score Range: 0.75 – 0.82  

---

## ⚙️ How to Run Locally

1. Clone the repository:

```
git clone https://github.com/aartinipurte-svg/insurance-cost-prediction.git
```

2. Navigate into the folder:

```
cd insurance-cost-prediction
```

3. Install dependencies:

```
pip install -r requirements.txt
```

4. Run the app:

```
streamlit run app.py
```
